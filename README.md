# Projeto Lógico de Banco de Dados do Zero - Oficina Mecânica 🛠️

Este projeto foi desenvolvido como parte do desafio da **Digital Innovation One (DIO)**. O objetivo é criar um esquema lógico para o contexto de uma oficina mecânica.

## 📋 Funcionalidades e Requisitos
O sistema permite o controlo total do fluxo da oficina, incluindo:
- Cadastro de Clientes e Veículos.
- Gestão de Ordens de Serviço (OS).
- Análises complexas através de queries SQL.

## 📂 Arquivos do Repositório
- `esquema_oficina.sql`: Script DDL com a criação das tabelas e chaves.
- `consultas_oficina.sql`: Script DML com as consultas que respondem aos requisitos do desafio.
